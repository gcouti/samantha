# samantha

This is a python program, it is a conversational agent that can integrate tools and build a lot of things. Its structure are build to have multiple conversational interfaces, like whatsapp, slack, or any integratable chat. 

We can develop tools to initetgrate to it and execute programs and plugins

# what it can do

* Find the best spot in your agenda and schedule a meeting
* Cria caracteristicas do seu contato com informações dos seus contatos, e caracteristicas do perfil pscicologico, lembretes, informações relevantes para conversas, etc
  * Exemplos:
    * Aniversário
    * Ultima vez que falou com a pessoa, manter a amizade
